☁️ **Especialista em:** Java, Kotlin, Spring Boot, Node.js e AWS  
🛠️ **Mentalidade:** Código limpo, arquitetura orientada a domínio e resiliência em primeiro lugar  
🔐 **Sistemas críticos:** Experiência em ambientes financeiros (B3, bancos, crédito, PIX)

- Soluções **serverless** na AWS (Lambda, etc.)
- Sistemas resilientes com foco em confiabilidade e escalabilidade
